## 典型方向清单（6 大方向）

适用于 AI 类旅行/生活助手项目：

1. **视觉美观** — 动画、微交互、字体、主题、骨架屏、动效
2. **UI 重新排版** — Landing 重构、分栏、详情页、移动端、PWA、i18n
3. **地图设计** — 路线动画、POI 增强、多目的地、截图分享
4. **LLM Prompt** — 上下文压缩、多轮迭代、对比输出、价格估算、A/B 测试
5. **知识库** — 数据扩展、数据库化、实时 API 接入、三方数据、偏好学习
6. **主动提问** — 多轮探测、反悔处理、隐性需求发现

## 典型 4 阶段划分

| 阶段 | 迭代范围 | 内容 |
|------|:--------:|------|
| Phase 1 基础 | 1-14 (28%) | 初始化、部署、核心功能、知识库雏形、Prompt 初版 |
| Phase 2 体验 | 15-30 (30%) | 视觉打磨、UI 重排、地图增强、移动端、PWA |
| Phase 3 智能 | 31-40 (20%) | Prompt 深度、知识扩展、个性化、主动探测 |
| Phase 4 生态 | 41-50 (22%) | 社交、运营、预订、会员、商业化 |

## 关键原则

1. **已完成迭代标 ✅**，让用户一眼看到进度
2. **人工介入项单独列出**，不与迭代混排
3. **Phase 1 可多于 14 次** — 取决于项目初始复杂度
4. **6 大方向可增减** — 但保持每个方向至少 3 次迭代
5. **下一迭代 = 当前未完成的第一个编号** — 创建后立即开始执行
