# Region Planner Design Spec（用户项目规范）

> 用于保持跨地区规划器视觉一致性。后续所有新地区项目均需遵循此设计规范。

## 整体布局

**三栏布局：**
| 栏位 | 角色 | 内容 |
|------|------|------|
| 左侧栏 | 城市/路线 + 预算 | 可滚动城市列表、路线选择器、天数/人数滑块、费用估算面板、商家收益面板 |
| 中间栏 | 主内容区 | Tab 切换：景点/酒店/餐厅/实用信息/增值服务 |
| 右侧栏 | 地图 | Leaflet 地图，高德/OSM/CartoDB 多源切换 |

## 配色方案

```css
:root {
  --bg0: #07070a;         /* 最深背景 */
  --bg1: #0b0b0d;         /* 面板背景 */
  --panel: rgba(255,255,255,0.05);  /* 半透面板 */
  --panel2: rgba(255,255,255,0.08);
  --stroke: rgba(255,255,255,0.1);
  --stroke2: rgba(255,255,255,0.15);
  --text: rgba(255,255,255,0.92);
  --muted: rgba(255,255,255,0.62);
  --muted2: rgba(255,255,255,0.42);
  --gold: #d6b46a;        /* 金色主色 — 奢华 */
  --gold2: #f2d190;       /* 亮金色高亮 */
  --accent: #90d5ff;      /* 蓝色交互色 */
  --radius: 14px;         /* 面板圆角 */
  --radius2: 20px;        /* 大卡片圆角 */
}
```

## 字体

```css
--serif: ui-serif, "Times New Roman", Georgia, serif;       /* 标题/展示 */
--sans: ui-sans-serif, system-ui, -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif;  /* 正文 */
```

- 衬线字体用于高端展示文字、金色高亮值
- 无衬线字体用于正文，中文优先级 PingFang SC → Microsoft YaHei

## 间距

- 面板圆角: 14px（标准）, 20px（大卡片）
- 卡片间距: gap: 4px（价格网格内）
- 面板间间距: 8px
- 分割线: border-top:1px solid var(--stroke)

## 常见陷阱

### 覆盖色块
左侧栏和中间栏上方曾出现半透明色块覆盖（z-index/CSS 伪元素问题）。确保面板无多余的 `::before`/`::after` 或定位重叠元素。

### 左侧栏滚动
左侧栏内容可能溢出，必须设置 `overflow-y: auto` 并确保正确高度约束。

### 标题换行
中间栏标题需强制单行显示，使用 `text-overflow: ellipsis; white-space: nowrap; overflow: hidden`。
