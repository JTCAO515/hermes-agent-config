## 关键决策

### 莫斯科→圣彼得堡：高铁替代自驾
650km 自驾需8小时，不适合9天行程。用 DRIVES 段 `alt` 字段标注萨普桑号高铁：
```
alt:"推荐高铁: 4小时, ¥450-800/人, 莫斯科→圣彼得堡萨普桑号"
```
预算函数在 classic 路线中自动加入高铁费用。

### 金环古城: 3座足矣
弗拉基米尔/苏兹达尔/谢尔盖耶夫镇各有特色（白石灰岩建筑/田园古镇/宗教圣地），
不必加雅罗斯拉夫尔等更远古城。三城总自驾 < 420km，适合嵌在莫斯科-圣彼得堡之间。

### 酒店选择: 覆盖各档位
莫斯科和圣彼得堡各4家五星（覆盖国际品牌+本土传奇如大都会酒店），
金环古城配4星精品和3星传统驿站，确保三个档次都能选到酒店。

### 餐厅: 强调文化体验
不仅列价格和菜系，每家写明历史背景和文化典故（如普希金咖啡馆的24小时营业、
修道院食堂的东正教斋戒食谱），让行程本身有故事。

## 遇到的问题

### JS 语法错误: `h.innerHTML+=;`
生成 HTML 时残留语法碎片，导致页面 JS 不可运行。虽然 curl 返回 200，
但浏览器会报错。解决方案：写入后立即用 `node -e "new Function(...)"` 做语法检查。
此陷阱已写入 SKILL.md 的「常见陷阱」节。
