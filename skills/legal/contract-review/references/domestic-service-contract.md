# 家政劳务合同陷阱清单

> 来源：2026-05-30 审阅《弘冠家政服务公司劳务合同》（甲方严国萍 × 乙方谷下凤，弘冠家政居间）

## 🔴 高风险（必须处理）

### 1. 介绍费「一律不退」— 显失公平

**典型条款：** *介绍费一律不退。*

即使甲方对服务不满意、需要更换人选，介绍费也不退。合同只承诺"提供候选人"，不保证满意。

**建议修改方向：**
- 按剩余服务期限比例退还介绍费
- 3个月内对服务不满意可全额退介绍费
- 或明确公司应承担的服务质量连带责任

### 2. 违约金对等条款 — 实际不对等

**典型条款：** *任何一方要求解除合同，需提前一个月通知对方，否则支付5500元违约金。*

表面对等，但家政行业乙方擅自离岗是高频风险。乙方走了追偿成本极高，而甲方违约乙方起诉反而更容易。

**建议修改方向：**
- 增加"乙方未经同意擅自离岗的，除支付违约金外，应赔偿甲方产生的临时保姆费用"

### 3. 乙方擅自离开 → 甲方承担「所有问题」

**典型条款：** *乙方擅自离开用户家，甲方必须在24小时内通知公司，否则乙方所出问题均由甲方承担责任。*

"乙方所出问题"范围无限——包括乙方离开后在别处发生的交通事故、刑事案件等。

**建议修改方向：**
- 限缩为"乙方擅自离开后发生的治安、刑事违法行为，由乙方自行承担全部法律责任，甲方不承担连带责任"
- 或不与24小时通知挂钩

### 4. 意外事故「承担一定经济责任」— 金额不明

**典型条款：** *造成本人或他人的意外事故，甲方应承担一定经济责任。*

"一定"太模糊。1万还是100万？

**建议修改方向：**
- 改为"甲方按过错比例承担相应责任，赔偿上限不超过已支付的服务费总额"
- 或明确以家政险赔付为上限

## 🟡 中风险（建议关注）

### 5. 人身损害免责条款 — 可能被认定无效

**典型条款：** 详细列举6种乙方自行承担的情形（违规操作、玩手机摔倒、隐瞒旧疾等）。

依据《民法典》第497条，提供格式条款一方不合理地免除或减轻自身责任的条款无效。甲方作为雇主负有基本安全保障义务。

**建议：**
- 增加"甲方应尽到安全提示义务"为前提
- 为乙方购买家政意外险（年保费约200-400元）
- 保险覆盖大部分风险，条款作为过错分配依据而非免责工具

### 6. 乙方赔偿责任无上限

**典型条款：** *因工作失误，造成的损失，均由自己负责。*

乙方月薪5500元，重大失误可能造成数万元损失，乙方赔不起，最终甲方扛。

**建议修改方向：**
- 改为"因乙方重大过失造成的直接损失，每月扣除金额不超过当月工资的20%"
- 建议甲方购买财产险

### 7. 公司责任缺位

公司只出现在收款条款，合同全文无公司违约责任。

**建议：**
- 要求增加"乙方提供的信息（身份、健康、无犯罪记录）由公司核实并承担连带责任"

## 🟢 提示注意

| # | 内容 | 说明 |
|---|------|------|
| 1 | 合同性质模糊 | 劳务合同 ≠ 劳动合同。若构成事实劳动关系，甲方可能承担社保和工伤责任。建议明确约定为劳务关系。 |
| 2 | 法定假期13天可调休 | "可根据实际情况调休"给了甲方操作空间，建议约定法定假期原则上当日休 |
| 3 | 无试用期 | 建议增加7-15天试用期，试用期内任一方可无责解除 |
| 4 | 无交接清单 | 建议附《家庭物品交接清单》双方签字，避免纠纷 |

### 8. 三方协议：公司、消费者、服务人员的三方权责不对等

**典型场景：** 甲方=家政公司（经营方）、乙方=消费者、丙方=服务人员。
- 乙方同时付给甲方服务费 + 丙方劳务费（年总支出可达7万+）
- 但甲方（公司）除了收款外，几乎不承担任何服务保障义务
- 丙方由甲方管理，但服务出问题时甲方说"这是丙方个人行为"

**消费者（乙方）的关键风险条款：**

1. **丙方离职后的"被架在空中"条款：** "丙方需要解除雇佣关系，需提前30天通知甲方。由甲方对接乙方进行人员更换及后续补充协议。"
   - 甲方只说"对接"，不承诺几天内到岗
   - 用户面临30天空窗期要自己找人带娃
   - **建议增加：** "甲方应在5个工作日内安排替代人员，逾期每日赔偿300元"

2. **续签由甲方单方决定：** "经甲方同意方可续签协议"
   - 乙方付了年费、用了一年习惯了，甲方一句不同意就得换公司
   - **建议删除：** "经甲方同意"，改为三方协商一致

3. **服务费退款公式缺失：** "甲方应退还乙方剩余的服务费"
   - 不写计算方式不写时限，甲方可以拖
   - **建议明确：** "按日折算，剩余服务费=总服务费×(剩余天数/365)，7个工作日内退还"

4. **被动解除权的"换人耐力赛"：** 需甲方先后调换3人仍不满意才能解约
   - 最短36天才能合法解约
   - **建议改为：** 调换2人即可，新人有3天试用评估期

## 消费者立场审阅方法论（2026-05-30 Session 新增）

> 以下框架来自用户明确要求"从消费者角度去审阅，最大程度保障消费者权益"。在审阅任何消费者与服务方之间的合同时，默认使用此方法论。

### 核心原则

1. **先算经济账**：消费者年总支出 = 服务费 + 劳务费/服务费。算出全年数字后，每个条款都问"这个条款在什么场景下会让消费者损失多少钱？"
2. **关注合同没写的内容**：公司方义务空白、服务中断补救缺位、退费机制缺失——这些比写得不好的条款更危险
3. **每个风险点给出谈判话术**：不只是指出问题，还要告诉消费者"可以怎么跟对方说"
4. **划分谈判优先级**：
   - **底线（不改不签）**：服务中断后的补人/退费/止损机制
   - **核心争取**：续签权、经济安全保障
   - **加分项**：服务质量监督、隐私保护等

### 消费者立场与平衡分析对照

| 维度 | 平衡分析（默认） | 消费者立场（优先） |
|------|----------|----------------|
| 续签条款 | "经甲方同意方可续签，建议三方协商" | "甲方可以单方面不让你续签，你投入的服务费和新人的适应成本就是他们的筹码——签之前改掉" |
| 退费条款 | "应退还剩余服务费，建议明确计算方式" | "退多少钱、多久退、按什么标准——全空着，甲方可以拖你3个月才退一半" |
| 服务中断 | "由甲方对接安排替换" | "服务人员走了你的生活断档，甲方只说'对接'不承诺几天补人——这期间你要自己扛" |
| 体检费用 | "乙方有权要求重新体检，费用由乙方承担" | "甲方有义务派健康合格的人，凭什么消费者为甲方的义务买单？" |
| 保险责任 | "由保险公司按规定赔偿" | "保险公司赔不赔、赔多少、免赔额多少——全没说，等于没说" |

### 消费者审阅的输出结构

```
## 📊 经济账：消费者年总支出 = ¥XX,XXX

## 🔴 底线条款（不改不签）
- 每条含：原文 → 消费者风险 → 经济损失场景 → 修改建议（含具体措辞）

## 🟡 核心争取条款
- 同上结构，但谈判优先级次之

## 🟢 加分项
- 锦上添花、可以放到"其他约定"中的补充条款

## ✅ 谈判策略
- 底线/核心/加分三级清单 + 每条的谈判话术
```

### 消费者审阅示例：经济损失场景计算

假设合同A：服务费4,800元/年 + 劳务费5,500元/月。

| 场景 | 经济损失 | 风险来源 |
|------|----------|----------|
| 服务人员第3个月离职，甲方拖30天补人 | 白付5,500元工资 + 4,800服务费不退 = **损失10,300元** | 补人时效缺失 |
| 服务不满意换4个人都不行，36天后才解约 | 36天无意义工资≈6,600元 | 换人上限太高 |
| 合同到期甲方不续签 | 损失4,800元服务费 + 新公司再交一次 | 续签单方决定权 |

## 三方协议特有检查清单

### 消费者角度加固版（2026-05-30 Session 新增）

#### 必须写进合同的补充条款

1. **服务人员离职补人时效**
   - 检查点：甲方是否承诺"X个工作日内安排替代人员到岗"？
   - 无此条则必须补充：5个工作日内补人，逾期日付300元补偿
   - 话术："一旦保姆走了，我家小孩没人带，这个空窗期你们必须给我一个时间承诺"

2. **续签保障**
   - 检查点：续签是否需要甲方"同意"？
   - 需要则必须删除"经甲方同意"，改为"乙方和丙方均同意续签的，甲方无正当理由不得拒绝"
   - 话术："我用了一年用得好好的，凭什么你甲方一句话就不让我续了？"

3. **退费公式**
   - 检查点：有没有写明"按日折算"或"按剩余月份/总月份"？
   - 无则补充：按日折算 + 7个工作日内到账
   - 话术："4,800元一年的服务费，如果半年后解约，到底退我多少？今天写清楚"

4. **不满意度门槛**
   - 检查点：解约需要换几个人才达标？
   - 超过2人则要求改为2人 + 每新人有3天试用评估期
   - 话术："换4个人才能证明你们不行？那我一个月工资打水漂谁赔？"

5. **保险保额和保单**
   - 检查点：保险保额多少？什么险种？保单能不能看？
   - 无则补充：不低于100万家政责任险，保单副本交消费者备案
   - 话术："你们说由保险赔，但你得告诉我保了多少、什么情况赔、什么情况不赔"

#### 消费者权益补丁（写进"其他约定"第七条）

```
1. 甲方应在5个工作日内安排替代人员到岗。逾期每日向乙方支付300元补偿金。

2. 乙方和丙方均同意续签的，甲方无正当理由不得拒绝。

3. 合同解除时，剩余服务费按日折算：应退金额 = 总服务费 × (剩余天数/365)。甲方应在7个工作日内退还。

4. 乙方对丙方不满意的，经调换2人仍达不到标准的，乙方有权解除合同。每新人有3个工作日试用评估期。

5. 甲方应提供丙方的无犯罪记录证明原件供乙方查验。丙方健康证应包括传染病筛查。

6. 甲方应购买不低于100万元保额的家政服务责任保险，保单副本交乙方。

7. 丙方因病或因事无法服务，应提前24小时通知。甲方应在12小时内安排临时替代人员。未能安排替代的，乙方有权按每日200元标准从服务费中扣除。

8. 乙方有权在家政服务区域安装监控设备，提前告知丙方即可。监控仅用于安全保障和纠纷处理。

9. 丙方对知悉的乙方家庭隐私承担保密义务，离职后仍有效。违反的，丙方和甲方承担连带赔偿责任。

10. "一般家务"的范围包括但不限于：[此处列明具体工作内容]
```

#### 原检查清单（保留）

- [ ] 丙方离职后，甲方补人有没有时效承诺？逾期有无赔偿？
- [ ] 续签是否需要甲方同意？消费者有没有续签保障？
- [ ] 退款公式是否明确定义？有无退款时限？
- [ ] 不满意换人最多几次？每次多久？
- [ ] 甲方作为经营方的服务保障责任是否明确约定？
- [ ] 消费者年总支出（服务费+劳务费）是否在合同中汇总列明？
- [ ] 丙方背景调查（无犯罪记录、传染病筛查）由谁负责？
- [ ] 保险（家政责任险）由谁购买？保额多少？保单消费者能否查验？
- [ ] 介绍费是否可退、退多少、谁决定
- [ ] 违约金对等还是偏向一方
- [ ] 人身损害免责条款的效力风险
- [ ] 公司的实质责任（收款vs服务责任对等）
- [ ] 合同性质是劳务还是劳动关系
- [ ] 有无试用期
- [ ] 保险责任是否明确
- [ ] 擅自离岗的法律后果范围
- [ ] 附件（健康证、身份证复印件）是否齐全
- [ ] 丙方有无请假/病假替代方案？谁补班？
- [ ] 消费者是否有权安装监控/摄像头？
- [ ] 消费者家庭隐私是否有保密条款保护？
- [ ] 服务内容是否有详细清单？还是笼统的"一般家务"？
