# Earnings Coverage

Three earnings research products, escalating in depth.

---

## A. Earnings Preview

Pre-earnings analysis before a company reports quarterly earnings.

1. **Gather context**: company, quarter, consensus estimates (revenue, EPS, segment metrics), earnings date/time, prior quarter guidance.
2. **Build key metrics framework**:
   - Financial: revenue, EPS, margins, FCF, forward guidance
   - Operational (sector-specific): SaaS (ARR, NRR, RPO), Retail (SSS, traffic), Industrials (backlog, book-to-bill), Financials (NIM, loan growth)
3. **Scenario analysis** (bull/base/bear with stock implications):

| Scenario | Revenue | EPS | Key Driver | Stock Reaction |
|----------|---------|-----|------------|----------------|
| Bull | | | | |
| Base | | | | |
| Bear | | | | |

4. **Catalyst checklist**: 3-5 metrics that will determine the stock's reaction.
5. **Output**: one-page preview with estimates table, metrics watchlist, scenarios, and trading setup (options-implied move).

## B. Earnings Preview (Single Company — Concise)

A shorter 4-5 page version of the preview focused on a single company. Same structure but less depth per section. Use when the user wants a quick take on one company's upcoming report.

## C. Full Earnings Analysis

Post-earnings deep-dive report covering:
- Revenue and EPS vs. consensus (with variance analysis)
- Segment performance
- Margin analysis (gross, operating, net)
- Balance sheet and cash flow highlights
- Guidance analysis (new vs. old vs. consensus)
- Key metrics and operating drivers
- Management commentary highlights (from call transcript)
- Revised estimates and valuation
- Bull/bear case post-print
