# 竞品分析

## 框架
1. **竞品矩阵**: 横轴(功能完整度) × 纵轴(价格/定位)
2. **功能对比表**: 核心功能逐项对比 ✅/⚠️/❌
3. **定价对比**: 基础版/专业版/企业版
4. **GTM策略**: 渠道/内容/社区/销售
5. **差异化机会**: 对手做不到但我们能做的
6. **威胁评估**: 对方可能进攻的方向
