# ljg-paper: 论文精读与转述方法论

将一篇学术论文转化为一个连贯的故事。核心信念：**读通一篇论文的标准，不是看懂，而是能讲给一个不懂领域的聪明人听，并让他能复述出来。**

- **服务对象**：非学术界的聪明人。
- **核心任务**：讲述论文核心命题（命题）的"一生"。
- **最终目标**：听众在听完后，能复述出关于论文的**五件事**：
  1. **困境**：主角面临的难题（具体例子）及该领域的研究水位（现状基线）。
  2. **命题**：作者大胆提出的核心猜想（一句话概括）。
  3. **论证骨架**：支撑命题的机制、证据、以及最反直觉的副发现。
  4. **内核**：听完故事后能带走的新视角（"原来还能从这个角度看"）。
  5. **检验**：该命题在现实中何处成立、何处失效，以及由此产生的可验证预测。

## 故事骨架：七拍故事弧

论文被分解为一个七拍的故事弧，这是全文的节奏和结构要求，而非死板的子标题：

1. **主角**：故事的主角是谁？开篇即引入。
2. **困境**：主角撞上了什么坎？不解决会怎样？
3. **旧路**：前人是如何尝试的？他们的解法被什么"两难"卡住？同时，**立一把尺子**：量化该领域的水位（现状水平，典型的进步幅度）。
4. **转折**：作者看到了什么别人没看到的东西？将其凝练为一句**命题**——大胆到让人想反驳的猜想。
5. **解法**：作者如何通过机制和证据来支撑其命题？这是论证骨架的主体。
6. **结局**：用两三个最有力的数字，**挂回旧路立的尺子上**称重，说明进步幅度。单独呈现最反直觉的副发现。
7. **内核**：故事结束后，听众能带走的新视角。不是复述结论，而是提供一个看待问题的全新方式。

**故事延伸**：故事讲完并不结束，命题还需要"走出论文"：
- **博导审稿**：论文本身是否站得住脚？
- **检验（生活测试）**：命题在现实世界中表现如何？既有命中点，也有边界（失效点）。
- **检验（押注未来）**：如果命题为真，未来应该观察到什么？（可被打脸的预测）

**关键红线**：将章节标题遮盖后通读全文，必须听起来像一个连贯的故事，而非独立的段落报告。

## 输出结构与执行步骤

生成的文档包含固定章节名，每个章节承载特定角色：

| 章节 | 角色 | 对应执行步骤 |
| :--- | :--- | :--- |
| **速读 (v6.1新增)** | 写在最后，放在最前。提供三个层次的压缩卡片 | 全文写完后再提炼 |
| **问题** | 搭台。让读者"亲历"困境、旧路撞墙、转折预告。一个例子贯穿始终 | 步骤2 |
| **翻译** | 命题登场，论证骨架展开。沿用同一个例子。包含五个必有的清单 | 步骤3 |
| **核心概念** | 故事道具。挑选3个最关键的、不讲就说不下去的概念 | 步骤4 |
| **洞见** | 内核。可以带走的新视角，说明还能用于何处 | 步骤5 |
| **博导审稿** | 从博导视角评价论文真伪、预设、方法、实验诚意 | 步骤6 |
| **检验** | 命题出门。生活测试和押注未来 | 步骤7 |
| **启发** | 听众的故事。行动指南（迁移、混搭、反转） | 步骤8 |

**速读卡（第9步）详细要求**：
- **一句话**：平实、可识别地概括全文做了什么。
- **大想法**：唯一可以带走的思想，核心视角。
- **只记三件事**：半年后也忘不掉的三个承重点（为何困境值得解、命题靠什么撑、带走什么新视角）。

## 关键写作原则

- **故事流贯到底**：章节间要有钩子，每段末句为下一段开口。
- **一个锚点撑全文**：从"问题"到"翻译"到"核心概念"，全程围绕最初的那个具体例子展开，不换地图。
- **推理外显**：模拟"想明白的过程"，而非展示"想明白的结果"。让读者感觉结论是自己想出来的。
- **变形替代定义**：用"把A变形成B"来解释关系，优于直接说明"A和B是什么关系"。
- **落点在能用**：给出"这意味着你可以\_\_\_\_"，而非"这让我们重新思考\_\_\_\_"。

## 红线

1. **口语检验**：像跟朋友讲故事一样。
2. **零术语**：先用大白话解释，再提术语名。
3. **短词优先**：能短则短。
4. **一句一事**：每句话只推进一步逻辑。
5. **具体**：名词可见，动词有力。
6. **开头给理由**：问题部分第一句就要吸引人。
7. **不填充**：删掉所有学术套话。
8. **信任读者**：正文中不重复结论（速读卡除外）。
9. **诚实**：有硬伤直接说，看不懂直接说。
10. **面向六个月后的自己**：确保术语和公式都被翻译成自然语言。
11. **外行优先于凝练**：该展开就展开，不要为了"短"而牺牲理解。
12. **故事一气呵成**：通读无标题版本，看是否还是一个流畅的故事。
