# 咨询级研究报告

## 报告结构
1. **执行摘要** (1-2页) — 核心发现与建议
2. **背景与方法** — 研究范围/数据来源
3. **市场分析** — 规模/趋势/驱动因素
4. **竞争格局** — 主要玩家/份额/差异化
5. **SWOT 分析** — 优势/劣势/机会/威胁
6. **战略建议** — 短期(0-6月)/中期(6-18月)/长期
7. **附录** — 数据源/方法论文档

## 输出格式
- 优先生成 Markdown → 可转 docx/pdf
- 关键数据用表格呈现
- 每个战略建议带：目标/行动/KPI/时间线
