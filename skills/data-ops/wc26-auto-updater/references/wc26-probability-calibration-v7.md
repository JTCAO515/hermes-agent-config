# WC26 概率校准与 EV 封顶规范 (v7.0.6)

## 核心问题
原始双变量泊松模型概率（如 37.1%）直接乘以市场赔率（5.68）产生荒谬 EV（110.7%+）。真实赌球不存在这种数字。

## 用户要求
"赌球不可能有胜率百分之百的情况。要有真实的胜率。"
"所有的预测类的赔率、成功率都可以模拟，但是数据要可推敲。"

## 校准公式
```python
market_implied = 1.0 / odds_val
cal_strength = 0.55  # 55% 模型权重, 45% 市场权重
cal_prob = prob * cal_strength + market_implied * (1 - cal_strength)
```

## 后处理 cap（每日推荐 + 量化信号）
| 指标 | 上限 | 理由 |
|------|------|------|
| ev_pct | 30.0% | 真实博彩顶级 EV 通常 <20% |
| edge_pct | 15.0pp | 超过 15pp 模型与市场差异不可信 |
| kelly_pct | 10.0% | 防止 Kelly 分数极端 |
| decimal_odds | 20.0 | 超高赔率市场流动性差 |

## EV 前端显示
用 `.toFixed(1)`，不用 `.toFixed(0)`。显示如 30.0% 而非 30%。

## 信号强度映射
| EV范围 | 等级 |
|--------|------|
| ≥25% | 5级 |
| ≥12% | 4级 |
| ≥6%  | 3级 |
| ≥3%  | 2级 |
| <3%  | 1级 |

## 因子评分（每信号独一画像）
| 因子 | 权重 | 计算公式 |
|------|------|---------|
| value | 35% | min(100, ev_pct*5+8) |
| risk | 20% | max(5, 100-strength*16) - 平局减15 |
| confidence | 20% | min(100, cal_prob*70+15+高概率加成) |
| momentum | 10% | min(100, kelly_pct*3+5) |
| diversity | 15% | 主/客 85, 平局 55 |
