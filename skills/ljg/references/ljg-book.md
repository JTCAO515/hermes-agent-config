# 拆书方法论：ljg-book 完整框架

将一本书解构为「它在认知史上挪动的那一步」。不是书评或摘要，而是对作者思想骨架的冷峻抽取。

## 核心定义与目标

- **本质**：不是读后感、章节摘要或赞美，而是把作者的骨架抽出来摆在桌上。
- **核心价值**：一本书的价值不在于它说了什么，而在于它「挪动」了什么——从旧共识（X）到新观点（Y）的差量（**delta**）。其余都是布景。

## 方法论骨架：五件事 + 一张图

此框架顺着一场认知挪动走一条线，由五件核心事和一张参考系图构成。

### 1. 核心问题
- **任务**：区分题目 (Topic) 和真正的问题 (Question)。
- **方法**：
  - 读导言/前言，找"我写这本书是为了……"。
  - 看作者甩不掉、反复回到的那个点。
- **测试**：将你找到的问题套用在同领域另一本书上，如果也成立，说明它太抽象，需要重写得更具体。

### 2. 共识基线
- **任务**：确定作者动笔前，这个问题上的「默认答案」是什么。
- **方法**：列出 2-4 个主要立场（流派/人物），各用一句话说明。必须标明其中哪个是当时的主流共识（教科书、常识）。
- **检验**：基线必须针对作者的不满。基线摆对了，作者会说："对，我就是冲这个来的"。

### 3. 独特洞见 + delta
- **任务**：找到作者带来的新东西，并量化它与基线的差量 (delta)。
- **洞见形态**：一个公式 (H = -Σp log p)、一个理论框架 (变异+选择+时间)、一个模型 (System 1/2)、一个概念 (反脆弱)。四选一。
- **delta 量化**：用一句话格式：「之前大家以为 X，作者说：不对，是 Y。」
- **delta 三档**：
  - **修正**：共识大体对，作者补了一块。
  - **倒转**：共识答反了，作者翻过来。
  - **换轴**：共识问的维度不对，作者换了参考系（威力最大）。
- **强制要求**：必须列出作者「独占的术语/列表/区分」（如 KK 的 12 动词、马克思的生产关系）。这是拆书的"指纹"。

### 4. 核心观点
- **任务**：提炼作者最终得出的结论——合上书，作者要你带走的那句话。
- **要求**：锋利、能立得住、能被引用、能反驳。10 年后还能想起来。不能是软话（"人性是复杂的"）或套话。

### 5. 精神内核
- **任务**：萃取出整本书的灵魂，形态五选一，三行内交付。
  - 一个取景框 (如：变异+选择+时间)
  - 一个模型 (如：System 1/2)
  - 一个洞见 (如："组织的命运不是被战略决定，是被结构决定。")
  - 一个概念 (如：反脆弱)
  - 一个金句 (如："The medium is the message.")
- **三检**：
  1. 10 年后只能记一件事，就是这件吗？
  2. 这件事跟前 4 段重复了吗？
  3. 没读过的人听了会不会想去翻原书？

## 核心工具：第 6 段 - 认知地图 (ASCII 参考系图)

将认知格局画成一张 ASCII 图，使 delta 成为图上可见的距离。

- **原理**：源于《千脑智能》，大脑通过参考系的位置来认识事物。
- **制图四步**：
  1. **提炼轴**：来自共识基线的分歧维度。两端各有极名。图注中点出作者的「轴外之物」（他不证自明的地基）。
  2. **钉位置**：流派 (●)、作者 (★)、旧共识 (文字) 都钉上去。每个点须有几何理由（谁挨着谁，谁对着谁，谁在原点）。
  3. **标 delta**：旧共识到作者的距离，方向与档位（修正/倒转/换轴）一目了然。
  4. **走两步，做预测**：用作者的框去预测书外的新位置，然后对账。能预测才叫掌握。
- **图例规矩**：
  - 宽度 ≤80 字符。
  - 图下必须有图注，解释每个点的位置、delta 和轴外之物。
  - **禁止装饰框**：画不出轴的图是无效的。

## 核心质量控制：灵魂三问 + 三道测试

### 灵魂三问
1. **零点问**：这个问题的默认答案是什么？
2. **位移问**：作者最独占的 move 是什么？（灵魂）
3. **落点问**：作者想让读者带走的那一句话是什么？

### 三道测试
1. **替换测试**：这段能套到同领域另一本书上吗？能 => 重写。
2. **指纹测试**：这段有几个本书独有的术语？0个 => 不合格。3+ => 好。
3. **delta 测试**：读者能一句话说出「之前大家以为 X，作者说 Y」吗？说不出 => 回去补基线。

## 写作规范与红线

- **风格**：冷静、简洁、犀利、直白。中文白话：汪曾祺、王小波、阿城那一路。
- **表达**：不夸、不贬、不替作者辩护、不学术腔。
- **硬性红线**：
  1. 核心观点不能软。
  2. 精神内核形态五选一，不重叠前文。
  3. 必须包含作者独占术语。
  4. 共识基线必须摆。
  5. delta 必须点名。
  6. ASCII 参考系图必须真画，能做预测。
  7. 全程反翻译腔和"切痕风"（如"这一刀"、"钉死"）。
